# osTicket
![image](https://raw.githubusercontent.com/yoyo82725/osTicket/master/osticket-supsys-sm.png)

<ul>
<li>使用osTicket為底開發</li>
<li>有統計圖表功能</li>

<li>客服部經常使用</li>
</ul>
